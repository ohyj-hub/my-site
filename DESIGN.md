# 디자인 가이드

이 문서는 팀 업무 안내 페이지의 디자인 기준을 정리한 파일입니다.
앞으로 디자인을 변경할 때 이 가이드를 기준으로 유지해 주세요.

---

## 전체 분위기

- **깔끔하고 신뢰감 있는 회사 소개 페이지** 느낌
- 복잡한 장식 없이 여백을 넉넉하게 사용
- 흰색 배경에 파란색(포인트)과 어두운 회색(강조)의 조합
- 모바일, 태블릿, PC 모두에서 잘 보이는 반응형 레이아웃

---

## 색상

| 이름 | 설명 | 사용처 |
|---|---|---|
| 흰색 (`white`) | 기본 배경 | 카드, 헤더, 메인 배경 |
| 밝은 회색 (`slate-50`) | 페이지 상단 그라데이션 시작 색 | 배경 그라데이션 |
| 진한 남색 (`slate-900`) | 강한 강조 색 | CTA 섹션 배경, 푸터 배경 |
| 포인트 파란색 (`blue-600`) | 핵심 강조 및 버튼 | 뱃지, 헤드라인 강조 텍스트, 문의하기 버튼 |
| 밝은 파란색 (`blue-50`) | 파란색 배경이 필요한 곳 | 팀 이름 뱃지 배경 |
| 중간 회색 (`slate-500`) | 본문 보조 텍스트 | 설명 문구, 카드 내용 |
| 진한 회색 (`slate-800`) | 카드 제목 등 | 업무 카드 제목 |
| 연한 회색 (`slate-100`) | 카드 테두리 | 업무 카드 외곽선 |

> **규칙:** 포인트 색상은 파란색(`blue-600`)만 사용합니다. 임의로 다른 색을 추가하지 않습니다.

---

## 글자 크기

| 요소 | 크기 | 설명 |
|---|---|---|
| 대표 헤드라인 (`h1`) | 48px (매우 큰 글씨) | 페이지 중심 문구 "함께 만드는 더 나은 제품" |
| 섹션 제목 (`h2`) | 30px (큰 글씨) | CTA 섹션의 "협업 또는 문의가 있으신가요?" |
| 카드 제목 (`h3`) | 18px (중간 글씨) | 업무 카드의 제목들 |
| 일반 본문 | 16px (기본) | 버튼 텍스트, 문의 안내 문구 |
| 카드 설명 | 14px (작은 글씨) | 카드 안의 상세 설명 |
| 헤더/뱃지 | 12px (매우 작은 글씨) | 상단 "Team Portal" 레이블, 뱃지 텍스트 |

> **규칙:** 글자는 충분히 크게 유지해 읽기 편하게 합니다. 카드 설명문(14px)이 최소 크기입니다.

---

## 여백 (간격)

- **페이지 좌우 여백:** 양쪽에 약 24px
- **최대 너비:** 전체 콘텐츠는 최대 1024px로 제한해 너무 넓어지지 않게 유지
- **섹션 간 여백:** 위아래 80px 이상 넉넉하게
- **카드 내부 여백:** 32px (카드 안쪽 텍스트와 테두리 사이 충분한 공간)
- **카드 간격:** 24px

> **규칙:** 여백은 줄이지 않습니다. 답답해 보이지 않도록 넉넉한 여백을 유지합니다.

---

## 버튼 모양 ("문의하기" 버튼)

- **모양:** 완전히 둥근 알약 형태 (pill shape)
- **색상:** 파란색 배경 (`blue-600`) + 흰색 텍스트
- **크기:** 좌우 32px 패딩, 상하 16px 패딩
- **글자:** 보통 굵기(semibold), 16px
- **그림자:** 아래쪽 방향의 파란 계열 부드러운 그림자
- **마우스 올렸을 때:** 조금 더 밝은 파란색으로 변함 (`blue-500`)

> **규칙:** 버튼은 항상 둥근 형태를 유지합니다. 각진 버튼으로 바꾸지 않습니다.

---

## 카드 (업무 소개 3개)

- **모양:** 모서리가 둥근 사각형 (`rounded-2xl` = 약 16px 반경)
- **배경:** 흰색
- **테두리:** 연한 회색 얇은 선
- **그림자:** 기본은 아주 연한 그림자, 마우스 올리면 살짝 진해짐
- **호버 효과:** 마우스를 올리면 카드가 위로 살짝 떠오름 (2px 이동)
- **배치:** PC에서는 3개가 나란히, 모바일에서는 1개씩 세로로

---

## 구조 (위에서 아래 순서)

```
[헤더] - 흰 배경, 팀 포털 레이블 + 이메일
[히어로] - 뱃지 + 큰 제목 + 부제목
[업무 카드 3개] - 흰 배경 카드 3개
[문의 CTA] - 어두운 배경 + 문의하기 버튼
[푸터] - 어두운 배경, 저작권 문구
```

---

## 변경 시 주의사항

1. 색상을 바꿀 때는 이 가이드의 색상 표에서 고릅니다.
2. 글자 크기를 줄이지 않습니다 (최소 14px).
3. 버튼의 둥근 모양을 유지합니다.
4. 여백을 너무 줄이지 않습니다.
5. 카드는 항상 3개를 유지합니다.
